Heroku 设置

类别 内容
前提 首先，知道只要有登陆账号和密码即可，然后，在任何 PC 或 Mac 上，下载一份Heroku Toolbelt，内含 Heroku Client，Foreman 以及 Git，点击安装
Login heroku login
SSH Keys 首先，要清楚，你随时可以在任何 PC 或 Mac 上，新生成一对 SSH 公钥和私钥，然后，通过 Heroku 命令来上传其中的公钥。如果本地已经有了一对，尤其是之前 Mac 安装过 Github 客户端，通常在个人账号根目录下属的 .ssh 文件包就有。 如果有，直接上传。 

heroku keys //查看 heroku 网站上已经有的 ssh public-key，此外，登陆 heroku 网站，点击 account 中也可以查看到，并且可以删除
heroku keys:add //把本地已经有的 public key 上传到 heroku 上
ssh-keygen -t rsa -f id_rsa //本地生成新的 ssh keys，其中，把 id 修改为你个人喜欢的 id 名称
本地 Clone 一份 heroku app heroku git:clone -a myApp，备注：这一步在 Mac 上操作出现问题，在 PC 上没有问题！
Push 这部分纯粹就是 git 命令。在安装 heroku 客户端时，已经设置了一个远程 Repo 名称为 heroku 

git status
git add .
git commit -m "...."
git push heroku master
Create 参考 heroku 在线教程：Getting started，按步操作即可。
